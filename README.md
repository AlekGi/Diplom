### Автоматизированноы тесты для веб-приложения МЭЛТОР

Были подготовлены следующие тесты:
1. Проверка авторизации
  * Проверка авторизации с пустыми полями логина и пароля
  * Проверка авторизации с валидными полями
  * Проверка авторизации с валидным логином и пустым полем пароля
  * Проверка авторизации с валидным паролем и пустым полем логина
  * Проверка авторизации с невалидным логином (!@#$) и валидным паролем
  * Проверка авторизации с невалидным паролем (!@#$) и валидным логином
2. Проверка добавления проекта из поиска закупок
  * Проверка добавления проекта из поиска закупок без добавления фильтров
  * Проверка добавления проекта из поиска закупок с добавлением поиска по ключу «Строительство»
  * Проверка добавления проекта из поиска закупок с добавлением поиска по Региону (городу) «Центральный ФО»
  * Проверка добавления проекта из поиска закупок с добавлением поиска по Региону (городу) «Центральный ФО» и ключом «Строительство»
3. Проверка работы календаря задач
  * Завершение текущей задачи и постановка следующей задачи из цепочки
  * Добавление новой задачи к проекту
  * Редактирование текущей задачи
